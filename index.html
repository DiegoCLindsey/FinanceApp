Cómo configurar Firebase
  1. 1Ve a console.firebase.google.com y crea un proyecto.
  2. 2Activa Authentication → Email/Password.
  3. 3Activa Firestore Database (modo producción).
  4. 4Copia las reglas de Firestore del archivo firebase/firestore.rules.
  5. 5En Configuración del proyecto → Tus apps → añade una app web → copia el objeto firebaseConfig.
  6. 6En Firestore crea la colección whitelist y añade un documento con tu email como ID.
🔐 Tus datos se cifran con AES-256 en tu dispositivo antes de llegar a Firebase. Ni el admin puede leerlos.
Cómo obtener un token de Dropbox
  1. 1Ve a dropbox.com/developers/apps e inicia sesión.
  2. 2Clic en Create app → Scoped access → App folder.
  3. 3Nombre de la app (ej: financeapp) → Create app.
  4. 4Pestaña Permissions → activa files.content.read y files.content.write → Submit.
  5. 5Pestaña Settings → sección OAuth 2 → Generate (access token). Copia el token.
✅ La app solo accede a su propia carpeta /Apps/financeapp/. No puede ver el resto de tu Dropbox.
